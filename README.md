# doctor-app
医疗系统

## 前端开发
项目基于gulp构建前后端不分离
```
yarn install
```
## 启动
```
yarn run server
```
### 后端使用
后端在使用时只用关心`css` `script` `images` `uploads`目录和`.html`文件
